# teachersweb Local Server

هذا المشروع هو موقع ثابت يحتوي على صفحات HTML و CSS.

## إعداد وتشغيل السيرفر

1. تأكد أن Node.js مثبت على جهازك.
2. افتح موجه الأوامر في المجلد `d:\WEBSITE\teachersweb`.
3. نفذ الأمر:

```bash
npm install
```

4. بعد الإكمال، شغّل السيرفر:

```bash
npm start
```

5. افتح المتصفح وتوجه إلى:

```text
http://localhost:3000
```
